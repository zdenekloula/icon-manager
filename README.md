Tool for svg icon sprite generating.

# What to do
## Install
```
npm install -g icon-manager
```

## Run
```
icon-manager
```

## Problems
### If you encounter a PhantomJS problem during your installation, use following command to install
```
npm install -g icon-manager --unsafe-perm
```
